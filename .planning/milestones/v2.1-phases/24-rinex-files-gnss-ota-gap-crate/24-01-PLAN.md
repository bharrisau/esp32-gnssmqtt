**Functions to implement:**

`fn write_obs_header<W: Write>(w: &mut BufWriter<W>, station: &str, approx_xyz: (f64,f64,f64), first_obs_utc: &DateTime<Utc>) -> std::io::Result<()>`
- Follow the exact header lines from RESEARCH.md Pattern 1 (all 9 header records)
- Each `writeln!` must produce a line where cols 61-80 contain the label
- Use `{:<60}` for the data field then append the label directly (total 80 chars before the newline)

`fn write_obs<W: Write>(w: &mut W, value: Option<f64>, lli: u8, ssi: u8) -> std::io::Result<()>`
- `Some(v)`: `write!(w, "{:14.3}{}{}", v, lli, ssi)` — exactly 16 chars
- `None`: `write!(w, "                ")` — exactly 16 spaces

`fn cnr_to_ssi(cnr: Option<f64>) -> u8`
- Per RTKLIB formula: `((cnr - 15.0) / 6.0 + 1.5).round().clamp(1.0, 9.0) as u8`; `None` returns 0

`fn pseudorange_m(obs: &Observation) -> Option<f64>`
- Returns `obs.pseudorange_ms.map(|ms| ms * SPEED_OF_LIGHT_M_PER_MS)`
- If pseudorange_ms is already full reconstruction (Task 1 Step 1), this is correct
- If rough_range_ms is stored separately, returns `Some((obs.rough_range_ms + obs.pseudorange_ms?) * SPEED_OF_LIGHT_M_PER_MS)` — adapt based on what Step 1 produces

`fn carrier_phase_cycles(obs: &Observation) -> Option<f64>`
- GPS: `obs.carrier_phase_ms.map(|ms| ms * GPS_L1_HZ / 1000.0)`
- GLONASS: return `None` (FCN not available; writes 16 spaces per STATE.md decision)
- Galileo/BeiDou: use GPS_L1_HZ as approximation for E1/B1 (1575.42 MHz); close enough for RTKLIB

`fn to_rinex_prn(c: Constellation, sv_id: u8) -> String`
- Gps -> `format!("G{:02}", sv_id)`
- Glonass -> `format!("R{:02}", sv_id)`
- Galileo -> `format!("E{:02}", sv_id)`
- BeiDou -> `format!("C{:02}", sv_id)`

`fn write_epoch<W: Write>(w: &mut BufWriter<W>, epoch_utc: &DateTime<Utc>, group: &EpochGroup) -> std::io::Result<()>`
- Write epoch header line per RESEARCH.md Pattern 2
- Handle >12 satellites with continuation line (32-space prefix)
- For each satellite, call write_obs three times: pseudorange_m (C1), carrier_phase_cycles (L1), cnr as Some(cnr) (S1)
- One writeln per satellite (3 obs = 3×16 = 48 chars, fits in 80)
