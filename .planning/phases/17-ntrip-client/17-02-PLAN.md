```rust
} else if t.ends_with("/ntrip/config") {
    match ntrip_config_tx.try_send(data.to_vec()) {
        Ok(_) => {}
        Err(TrySendError::Full(_)) => log::warn!("mqtt cb: ntrip config channel full — payload dropped"),
        Err(TrySendError::Disconnected(_)) => {}
    }
}
```
Place before the final `// All other topics: silently ignored` comment.

**Change B — subscriber_loop /ntrip/config subscription:**
In `subscriber_loop`, inside the `Ok(mut c) =>` match arm after the existing `/log/level` subscribe call, add:
```rust
let ntrip_config_topic = format!("gnss/{}/ntrip/config", device_id);
match c.subscribe(&ntrip_config_topic, QoS::AtLeastOnce) {
    Ok(_) => log::info!("Subscribed to {}", ntrip_config_topic),
    Err(e) => log::warn!("Subscribe /ntrip/config failed: {:?}", e),
}
```
QoS::AtLeastOnce so retained config is re-delivered on broker reconnect (required for NTRIP-02 persistence across device restarts when caster config is set via MQTT).
